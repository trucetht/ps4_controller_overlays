# Pastel PS4 GamePad Viewer Skin

This repository contains an Edit CSS skin for the PlayStation 4 / DualShock 4 controller in GamePad Viewer.

## Colors

- Light-pink faceplate
- White face buttons
- Powder-blue thumbsticks
- Black D-pad

## Hosted CSS URL

After enabling GitHub Pages, the URL will be:

```text
https://YOUR-USERNAME.github.io/ps4-pastel-controller/ps4-pastel-controller.css
```

Replace `YOUR-USERNAME` with your GitHub username. If you choose a different repository name, replace `ps4-pastel-controller` as well.

## GamePad Viewer

Use the CSS as an **Edit CSS URL**, not as a complete Custom CSS replacement. Select the **PlayStation 4 White** base skin.
